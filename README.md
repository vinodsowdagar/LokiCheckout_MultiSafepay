# Yireo_LokiCheckoutMultiSafepay

**This is an add-on package to the LokiCheckout. It is only a dummy to kickstart new modules easily.**

Current status: Draft

## Installation
Install this package via composer (assuming you have setup the `composer.yireo.com` repository correctly already):
```bash
composer require yireo/magento2-loki-checkout-multi-safepay
```

Next, enable this module:
```bash
bin/magento module:enable Yireo_LokiCheckoutMultiSafepay
bin/magento setup:upgrade
```
